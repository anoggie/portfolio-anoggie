$ Projetos

Extrator de dados de campos de logs

Programa criado em Python com o intuito de auxiliar nas análises de logs do Apache. O usuário insere o arquivo .txt dos logs, e extrai os campos que deseja. As informações são guardadas em um arquivo .csv.

Calculadora de sub-rede IPv4

Script desenvolvido para calcular sub-redes IPv4, permitindo que o usuário obtenha o intervalo de IPs e o número total de sub-redes disponíveis a partir de uma rede base.

Sniffer de Pacotes

Script para captura e análise de pacotes de rede em tempo real. O objetivo é permitir que o usuário monitore o tráfego de dados em uma rede, incluindo requisições HTTP, tráfego DNS e protocolos como TCP e UDP.
